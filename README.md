# index-hkjc-pages

本仓库用于归档和发布多个独立的 HTML 页面。页面内容不针对特定网站或域名，仅作为通用前端页面的静态存档与分发用途。

## 项目简介

index-hkjc-pages 是一个静态页面集合仓库，旨在保存和展示若干独立的 HTML 文件。这些页面可能包含演示、工具、说明或其它前端内容，所有页面均以纯静态形式存放，无需后端支持。

## 目录说明

```
.
├── pages/           # 存放各个独立 HTML 页面
├── assets/          # 公共资源文件（CSS、JS、图片等）
├── index.html       # 仓库主页（可选）
└── README.md        # 本文件
```

- `pages/` 文件夹内每个子目录对应一个独立页面，包含该页面所需的 HTML 及相关资源。
- `assets/` 用于存放多个页面可能共用的样式、脚本或媒体文件，方便统一管理。

## 页面归档说明

- 每个页面均为独立完整的 HTML 文件，可直接在浏览器中打开。
- 所有页面均不包含动态后端逻辑，不依赖外部服务。
- 页面内容可能涉及不同主题或功能，彼此之间无直接关联。
- 归档仅用于保存和展示，不保证页面内容长期更新或维护。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立页面，或修复已有页面中的问题。
- 新增页面请放入 `pages/` 下对应目录，并确保资源路径正确。
- 请勿在页面中包含任何具体域名、网址、营销承诺或诱导点击内容。
- 本仓库仅作为静态页面存档使用，不承担页面内容以外的任何责任。
